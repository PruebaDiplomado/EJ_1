# Hola Mundo – HTML Básico

Este proyecto es un ejemplo básico de una página web creada con **HTML5**. Su objetivo es servir como práctica inicial para comprender la estructura fundamental de un documento HTML.

## Descripción

La página muestra dos encabezados en pantalla:

- Un título principal con el texto **“Hola Mundo”**
- Un subtítulo con el texto **“Cruel Cruel”**

Además, el archivo está preparado para enlazar una hoja de estilos CSS y un archivo JavaScript externos.

## Estructura del archivo

El documento HTML incluye:

- Declaración `<!DOCTYPE html>` para HTML5  
- Etiquetas semánticas básicas: `<html>`, `<head>`, `<body>`
- Metadatos para compatibilidad y diseño responsivo
- Enlaces a:
  - `main.css` (estilos)
  - `main.js` (scripts)
